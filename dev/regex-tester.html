O que são expressões regulares?

Regex (expressões regulares) são padrões usados para encontrar, validar ou substituir texto. São usadas em programação, editores de código e ferramentas de busca para encontrar padrões complexos em strings.

Flags

g — encontra todos os matches (global). i — ignora maiúsculas/minúsculas. m — trata cada linha como início/fim de string.
