JSON → CSV

Cole um array JSON e converta para CSV. Abre direto no Excel e Google Sheets.

Como funciona

  1. Cole um array JSON (lista de objetos com as mesmas chaves).
  2. Escolha o separador — use ponto e vírgula se for abrir no Excel em português.
  3. Marque "BOM UTF-8" para garantir que acentos apareçam corretamente no Excel.
  4. Clique em Converter e baixe o .csv.

Dicas de uso

Recebeu dados de uma API em JSON e precisa analisar no Excel ou compartilhar com alguém que não sabe programar? A conversão para CSV deixa tudo pronto para abrir direto em qualquer planilha.

  • Se for abrir no Excel em português, escolha ponto e vírgula como separador — o Excel BR usa ; por padrão ao importar CSV.
  • O conversor achata objetos aninhados automaticamente: {"endereco": {"cidade": "SP"}} vira uma coluna chamada endereco.cidade.
  • Marque a opção BOM UTF-8 para garantir que acentos e caracteres especiais apareçam corretamente ao abrir no Excel — sem essa opção, ç e á ficam corrompidos.

💡 Próximo passo: Antes de converter, valide e formate o JSON com o JSON Formatter — JSON mal formatado ou com erros vai interromper a conversão.
