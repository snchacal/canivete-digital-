Formatador de CSS

Formate e organize seu CSS com indentação correta.

Para que serve formatar CSS?

Código CSS minificado ou mal indentado é difícil de ler e manter. O formatador adiciona quebras de linha e espaços no lugar certo, deixando cada propriedade em sua própria linha e respeitando a hierarquia dos seletores.
