Converter maiúsculas e minúsculas

Escolha um modo e o texto é convertido na hora.

Modos disponíveis

  1. MAIÚSCULAS — converte tudo para caixa alta.
  2. minúsculas — converte tudo para caixa baixa.
  3. Title Case — primeira letra de cada palavra em maiúscula.
  4. Sentence case — primeira letra de cada frase em maiúscula.
  5. Alternado — alterna entre maiúscula e minúscula em cada letra.
  6. Invertido — inverte o case de cada letra.

Dicas de uso

Converter o case de um texto manualmente é tedioso e sujeito a erros — especialmente em listas longas de nomes, títulos ou siglas. Com essa ferramenta você padroniza qualquer texto em segundos.

  • "Title Case" é o padrão para títulos de artigos, posts de blog e nomes de produtos em inglês — cada palavra com inicial maiúscula.
  • "MAIÚSCULAS" é ideal para siglas e acrônimos. "minúsculas" é o formato mais usado para normalizar dados antes de salvar em banco de dados.
  • Ao importar dados de planilhas com nomes de clientes, use "Capitalizado" para garantir consistência antes de enviar e-mails personalizados.

💡 Próximo passo: Depois de ajustar o case, use o Removedor de espaços extras para limpar espaços duplos que podem ter vindo junto com o texto copiado.
