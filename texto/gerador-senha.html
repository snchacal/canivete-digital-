Gerador de senha

Crie senhas fortes e aleatórias. Tudo gerado localmente — nada é enviado para nenhum servidor.

Clique em "Gerar" para criar uma senha

Dicas de segurança

  1. Use pelo menos 16 caracteres para senhas importantes (banco, e-mail).
  2. Ative símbolos para aumentar muito a complexidade da senha.
  3. Nunca reutilize a mesma senha em sites diferentes.
  4. A geração é 100% local — nenhuma senha é armazenada ou enviada.

Dicas de uso

Uma senha forte é a primeira linha de defesa de qualquer conta. O gerador cria combinações aleatórias que seriam praticamente impossíveis de adivinhar — diferente de datas de nascimento ou nomes de pets.

  • Use no mínimo 16 caracteres com letras maiúsculas, minúsculas, números e símbolos para contas importantes como e-mail e banco.
  • Nunca reutilize senhas: se um site vazar sua senha, todas as outras contas ficam expostas. Gere uma senha diferente para cada serviço.
  • Guarde as senhas em um gerenciador como Bitwarden (gratuito e open source) ou 1Password — decorar senhas fortes é impossível.
