Dicas de resolução

  1. 96 dpi — suficiente para exibição na web, arquivo menor.
  2. 192 dpi — boa qualidade para visualização e uso geral.
  3. 288 dpi — alta qualidade, ideal para imprimir ou usar em apresentações.

Dicas de uso

Converter páginas de PDF em imagens é útil para criar thumbnails, extrair gráficos ou compartilhar conteúdo em plataformas que não abrem PDF diretamente, como Stories e apresentações.

  • Quanto maior a resolução escolhida, melhor a qualidade da imagem — mas maior o arquivo. Para web use 96–150 DPI; para impressão use 300 DPI.
  • Para extrair apenas uma slide ou gráfico específico, divida o PDF primeiro para isolar a página desejada, depois converta.
  • Imagens geradas de PDFs com texto tendem a ficar com texto legível mesmo em resoluções menores — ótimo para criar previews de documentos.

💡 Próximo passo: As imagens geradas costumam ser pesadas. Use a ferramenta de compressão de imagem para reduzir o tamanho antes de publicar ou enviar.
