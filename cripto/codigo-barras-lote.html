Código de Barras em Lote

Gere vários códigos de barras de uma vez e baixe todos em um arquivo ZIP.

Gerar sequência

Dicas de uso

O gerador em lote processa todos os códigos no browser, sem limite de quantidade — apenas o desempenho do seu computador pode ser um fator para listas muito grandes.

  • Para EAN-13, informe apenas os 12 primeiros dígitos — o 13º (dígito verificador) é calculado automaticamente.
  • Use a função "Números sequenciais" para gerar etiquetas de estoque, patrimônio ou controle interno em sequência.
  • O ZIP contém um arquivo PNG por código, nomeado com o valor do código — fácil de identificar e importar em outros sistemas.

💡 Próximo passo: Para um único código com mais opções de personalização, use o Gerador de Código de Barras 1D.
