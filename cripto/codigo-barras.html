Gerador de Código de Barras 1D

Gere EAN-13, Code 128, UPC-A e outros formatos. Baixe em PNG.

Qual formato usar?

  • EAN-13: Padrão para produtos vendidos no varejo brasileiro. Exige exatamente 12 dígitos (o 13º é calculado automaticamente).
  • Code 128: Formato geral — aceita letras, números e símbolos. Ideal para etiquetas internas e logística.
  • ITF-14: Para caixas e unidades de logística. Derivado do EAN-13 com dígito adicional.
  • UPC-A: Padrão norte-americano, comum em produtos importados. 12 dígitos.

Dicas de uso

Códigos de barras 1D são lidos por leitores laser e de câmera em PDVs, estoques e portarias. A escolha do formato correto é essencial para compatibilidade com o sistema que vai fazer a leitura.

  • Para produtos vendidos em supermercados brasileiros, use EAN-13 — é o padrão obrigatório da GS1.
  • Imprima com resolução mínima de 300 DPI para leitura confiável. Tamanho mínimo recomendado: 37,29mm × 26,26mm para EAN-13.
  • Sempre verifique o código gerado com um leitor real antes de imprimir em lote.
