Gerador de CPF / CNPJ

Gere números válidos para testes de desenvolvimento.

⚠️ Apenas para testes. Os números gerados são matematicamente válidos mas não pertencem a pessoas ou empresas reais. Usar CPF/CNPJ falso em transações é crime.

Como usar

  1. Escolha entre gerar CPF ou CNPJ.
  2. Selecione a quantidade de números (até 20 por vez).
  3. Clique em Gerar e copie os números para usar nos testes do seu sistema.

Dicas de uso

Os CPFs e CNPJs gerados por esta ferramenta passam na verificação matemática dos dígitos verificadores, mas não pertencem a nenhuma pessoa ou empresa real. São válidos somente para testes de desenvolvimento.

  • Use em formulários de cadastro durante o desenvolvimento para testar campos de validação sem usar dados reais de terceiros.
  • Nunca use números gerados para cadastros em sistemas reais, tentativas de fraude ou qualquer fim além de testes — isso constitui crime.
  • A validação matemática usa o algoritmo módulo 11 da Receita Federal — o mesmo usado em produção nos sistemas do governo.

💡 Próximo passo: Validador de CPF / CNPJ — valide os números gerados para confirmar que os dígitos verificadores estão corretos antes de usar nos testes.
